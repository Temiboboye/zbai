## üéØ How to Use

### **Option 1: Open in Browser**
1. Double-click `index.html` to open in your default browser
2. Use arrow keys or on-screen buttons to navigate
3. Press `F` for fullscreen mode
4. Press `P` to print/export to PDF

### **Option 2: Present from Browser**
1. Open `index.html` in Chrome, Firefox, or Safari
2. Press `F11` (Windows) or `Cmd+Shift+F` (Mac) for fullscreen
3. Navigate with arrow keys or click buttons
4. Press `Esc` to exit fullscreen

### **Option 3: Export to PDF**
1. Open `index.html` in Chrome
2. Press `Cmd+P` (Mac) or `Ctrl+P` (Windows)
3. Select "Save as PDF" as destination
4. Set margins to "None"
5. Enable "Background graphics"
6. Click "Save"
